Итоговый файл для запуска: psid_wideband_1e-5.py. В нем содержится кейс для разделения упрощенного OFDM-сигнала с использованием матрицы Адамара и, затем, методом MRC. Но если в других исходниках, приведенных здесь, вообще либо моделируется не многочастотный сигнал, либо BER измеряется в других пределах, то последний код с использованием метода Монте-Карло измеряет BER порядка 1e-5.

При преобразовании цифрового сигнала в аналоговую форму возникают сигналы за пределами основной зоны Найквиста, которые обычно фильтруются, но в данной работе показана возможность целенаправленно использовать их. Если на стороне передачи принципиальная возможность уже закладывается производителями современных ЦАП (генерация сигнала в зоне Найквиста M>1), то на стороне приема алиасинг таких зон с неотфильтрованным сигналом приводит к интерференции сигналов, обладающих случайными фазами после прохождения канала связи. Поэтому предложен особый метод построения приемника для реализации возможности использования эффекта алиасинга с пользой в системах связи с OFDM-сигналами.
